# Сайт бухгалтера (ООО/ИП для ресторанов)

Это простой статичный сайт: откройте `index.html` в браузере — он сам подгрузит `styles.css` и `script.js`.

## Что поменять
- В `index.html` замените плейсхолдеры телефона/почты/города.
- В форме укажите реальный email получателя в атрибуте `data-email`.
